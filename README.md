# Библиотека статей

Сюда для сохранности копируются полезные статьи про мастерение ролевых игр и играние в них. На сами статьи и их авторов даются ссылки.
Цель: 
- не потерять полезные материалы в случае их случайного удаления;
- создать удобное пространство для навигации по разным материалам;
- создать единое пространство, которое можно пересылать своим товарищам.

Если вы хотите, чтобы вашу статью добавили сюда — напишите Лене Шпрингер в телеграм @lshpringer или Вконтакте https://vk.com/lshpringer.
